#  GuitarEik - Your Ultimate Guitar Companion

**GuitarEik** သည် ဂစ်တာတီးခတ်သူများအတွက် သီချင်းစာသားများ၊ Chord များနှင့် Artist အချက်အလက်များကို တစ်နေရာတည်းတွင် အလွယ်တကူ ရှာဖွေနိုင်ရန် တည်ဆောက်ထားသော Mobile Application တစ်ခုဖြစ်ပါသည်။ ဤ App ကို **Flutter** နှင့် **Spring Boot** (Backend) တို့ကို အသုံးပြု၍ Professional ဆန်စွာ တည်ဆောက်ထားပါသည်။

## Features (ပါဝင်သော လုပ်ဆောင်ချက်များ)

* **Chord Transposer:** သီချင်းများကို မိမိတီးခတ်လိုသည့် Key အလိုက် အလွယ်တကူ ပြောင်းလဲကြည့်ရှုနိုင်ခြင်း။
* **Artist & Album Directory:** မိမိနှစ်သက်သော အနုပညာရှင်များနှင့် Album အလိုက် သီချင်းများကို စနစ်တကျ ရှာဖွေနိုင်ခြင်း။
* **Favorite List:** မိမိအကြိုက်ဆုံး သီချင်းများကို သိမ်းဆည်းထားနိုင်ခြင်း။
* **Search Functionality:** သီချင်းအမည် သို့မဟုတ် အနုပညာရှင်အမည်ဖြင့် အမြန်ဆုံးရှာဖွေနိုင်ခြင်း။
* **Responsive UI:** Dark Mode နှင့် Light Mode နှစ်မျိုးလုံးတွင် ကြည့်ကောင်းသည့် ခေတ်မီ Design။

##  Installation for Developers

ဤ project ကို စမ်းသပ်လိုပါက:

1.  **Repository ကို Clone လုပ်ပါ:**
    ```bash
    git clone [https://github.com/brownsley/guitar-eik-flutter.git](https://github.com/brownsley/guitar-eik-flutter.git)
    ```
2.  **Project ထဲသို့ သွားပါ:**
    ```bash
    cd guitar-eik-flutter
    ```
3.  **Dependencies များ ရယူပါ:**
    ```bash
    flutter pub get
    ```
4.  **Application ကို Run ပါ:**
    ```bash
    flutter run
    ```
